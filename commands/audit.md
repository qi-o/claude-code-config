# Claude Command: Audit

对当前代码库进行全面质量审计。

## 触发方式

```
/audit
```

## 执行步骤

### 1. 代码复杂度分析
- 检查函数长度（目标 < 50 行）
- 检查圈复杂度
- 检查嵌套深度（目标 < 3 层）

### 2. 代码重复检测
- 查找重复/相似代码块（3 次以上）
- 标记可提取为共享函数的部分

### 3. 安全审计
- 检查硬编码 secrets（API keys, passwords）
- 检查未验证的用户输入
- 检查 SQL 注入风险

### 4. 测试覆盖检查
- 识别无测试覆盖的关键代码
- 检查测试质量（不是只有空测试）

### 5. 输出格式

```markdown
## 审计报告

### 🔴 严重问题
- [文件:行号] 问题描述

### 🟡 建议改进
- [文件:行号] 建议描述

### 🟢 良好实践
- 发现的良好模式
```

## 注意事项

- 只报告问题，不自动修复
- 按严重程度排序输出
- 提供具体文件位置和行号
